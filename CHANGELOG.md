# Changelog

## 0.5.0 - Release candidate

- Added task contracts, explicit task phases, completion verification and bounded repair feedback.
- Added advanced verifier checks, Python/JavaScript/TypeScript semantic indexing and professional evaluation contracts.
- Added run locking, garbage collection, lineage inspection, static reports and optional OTLP export.
- Added versioned seccomp/AppArmor policies, SBOM and vulnerability-scan release gates.
- Added a 100-task versioned professional evaluation suite and a 30-task local-model baseline. The measured local baseline is 40% and therefore keeps 0.5.0 in release-candidate status; cloud-provider gates remain unexecuted without credentials and explicit cost authorization.

## 0.4.0 - 2026-07-16

- Added Docker and explicitly degraded local execution backends.
- Added provider retries, request correlation, and runtime budgets.
- Added incremental workspace indexing and performance benchmarks.
- Added environment diagnostics, structured release checks, and live Ollama evaluation support.
